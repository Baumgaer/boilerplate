Class DeepTypedSchema<T>Abstract

Provides basic functionality for schemas with nested Types and constraints like "isRequires", "min" or "max". Owner and raw type might be set after construction because usually this schemas belong to a model which might be still under construction after this schema is constructed. So make sure to call the methods "setOwner" and "awaitConstruction".
