Class Schema<T>Abstract

Implements basic functionality for schema validation and defines necessary methods as a contract for subclasses. The schema type might be constructed asynchronously which is the reason for the existence of the method awaitConstruction. Make sure to await its result on every construction.

Type Parameters

  • T extends ModelLike

    The model where the schema of the model belongs to
